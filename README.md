Проєкт Кухарія створено в межах лабораторної роботи №2 «Створення структури веб-проєкту».
Сайт демонструє базову структуру вебресурсу, що складається з головної сторінки, інформаційних розділів, сторінки користувача та форми входу й реєстрації.

Структура проєкту включає теку /images із зображенням, /css із файлом стилів style.css, /js зі скриптом script.js, а також HTML-сторінки: index.html, about.html, contacts.html, recipe.html, account.html і login.html.

Для розробки використано технології HTML5, CSS3, JavaScript та систему контролю версій GitHub.
HTML та CSS код пройшли перевірку через W3C Validator, а зображення оптимізовано для швидкого завантаження.
