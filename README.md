У проекті використано Flexbox для побудови гнучких компонентів і вирівнювання.
Сітка рецептів і двоколонкові блоки на сторінці рецепта можна створити за допомогою CSS Grid.
Адаптивність досягається за допомогою медіазапитів.
Підключення окремих CSS-файлів за допомогою @import дозволяє організовувати стилі модульно.
Додано адаптивне бургер-меню з використанням CSS-трансформацій і анімацій. У проєкті також реалізовано глобальну типографіку та уніфіковані стилі для заголовків, тексту й інтерфейсних елементів

Виконавці: Катерина Богачова, Бубало Ганна, Животок Дар’я
